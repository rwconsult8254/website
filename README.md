# rogerwibrew.com

Personal site and blog for Roger Wibrew — chemical engineer working at the intersection of process engineering, simulation, and AI-enabled tools.

Built with [Astro](https://astro.build/).

## Topics

- Process simulation and thermodynamics
- AI/LLM applications in engineering workflows
- Interactive web-based engineering tools

## Development

```bash
npm install    # Install dependencies
npm run dev    # Local dev server at localhost:4321
npm run build  # Production build to ./dist/
```
